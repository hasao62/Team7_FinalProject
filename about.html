About Tango

The Tango bot was developed as a final project by Team 7 for ITP 258, Systems Development Project. This bot uses Discord's bot API to allow users to use text to speech functionality. This bot is hosted off of a Raspberry Pi which also hosts the database that stores the commands, user login, transcripts, and audio logs.

About the commands

  • !help

    • Use this command to view the help messages.

  • !join

    • This command followed by a voice channel name will allow the bot to join the desired channel.

  • !tts

    • Use the tts command followed by the desired speech to allow the bot to communicate into the voice channel.

  • !transcribe

    • This command records and posts a transcript of user speech into designated text channel.

  • !leave

    • This command takes the bot out of the voice channel.
